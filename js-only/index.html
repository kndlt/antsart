Controls:
[Space] - Pause physics
[2] - Toggle Line View
[3] - Toggle Oval View
[4] - Toggle Rescreen
[5]/[6] - Increase/Decrease Oval Size
[H] - Limit Paint
Scroll - Zoom, Drag - Pan

URL Parameters:
numPromisers, expansionPixel, speed
firstRatio, halfsize, zoom
lineView, ovalView, rescreen
limitPaint, paintLimit, spacepressed
width, height
Example: #numPromisers=5000&speed=5
